Bootcamp 2021: Generative Ball Art

The first programming project that I have done at City University, concluding the 2 weeks of Programming Bootcamp 2021. This Processing project that I have created demonstrates the understanding of the use of variable x position, using randomness for selecting the random colours in the specific balls and changing the colour while the balls rotates, using rotatiting with radians, PushMatrix and PopMatrix, KeyPressed, using the translate, and the framerate to control the speed of the rotation of the balls. I created the generative ball art in which the balls rotates and enlarges, when the mouse is positioned to X position. In addition the balls change the colour while rotation and when the mouseX is pressed and at the same time it increases the numbers of the balls appeared on the screen.
